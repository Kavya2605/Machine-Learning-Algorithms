# Machine Learning

This repository contains implementations and study materials for core Machine Learning concepts.
It includes examples of supervised learning, unsupervised learning, and model deployment workflows using Python and popular ML libraries.

## Topics Covered

### Supervised Learning

Supervised learning involves training a model using labeled data where the input features and the corresponding target outputs are known.

Algorithms included:

* Linear Regression
* Logistic Regression
* Decision Trees
* Random Forest
* Support Vector Machine
* K-Nearest Neighbors

Applications:

* Classification problems
* Regression problems
* Predictive modeling

### Unsupervised Learning

Unsupervised learning focuses on discovering patterns and relationships in data without labeled outputs.

Algorithms included:

* K-Means Clustering
* Principal Component Analysis (PCA)

Applications:

* Clustering
* Dimensionality Reduction
* Pattern Discovery

### Model Deployment

Model deployment is the process of making trained machine learning models available for real-world applications.

Tools and frameworks used:

* Flask
* FastAPI
* Docker
* REST APIs

Deployment tasks:

* Saving trained models
* Creating prediction APIs
* Integrating models into applications

## Technologies Used

* Python

* Pandas
* Scikit-learn
* TensorFlow / PyTorch
* Flask / FastAPI
